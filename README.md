# ecommerce-mini-project

React + Typescript Assignment –
E-commerce Mini Project
Project Objective
Build a small E-commerce application using React.js where you will practice:
● Components
● API consumption
● Routing
● State management
● Type & Interfaces
Pages You Must Create
1. Home Page
Create a homepage with:
● Header
● Top banner section
● Categories section
● PLP Product list
● List of Add to cart product list render below PLP
● Footer
Categories to show:
● Laptop
● TV
● Camera
● Mobile
● Accessories
2. PLP – Product Listing Page
○ This page must fetch product data from an API.
○ You are required to consume the below API:
API Url:
https://dummyjson.com/products
If any concern check this : http://dummyjson.com/docs
Navigation Requirement:
● When a user clicks on a product card, the user must be redirected to the PDP
(Product Details Page)
3. ADD TO CART products
When the user clicks “Add to Cart”:
✅ The product must be added to a Cart List section on the same page
✅ Show the product in a list with:
● Product Name
● Price
● Quantity input box
● Total price for that product
● Delete icon/button
Note:
● Product should remove if user click on delete button
● User click on quantity + or - button accordingly price changes.
4. PDP – Product Details Page
Create a PDP (Product Details Page) that shows the complete details of the selected product:
● Large Product Image
● Product Name
● Product Price
● Product Description
● Quantity selector
● Buttons:
○ Add to Cart to add product in rendered product list component
Add Back button return to Home page from PDP
The product information must be shown dynamically based on the product selected from the
PLP page.
Functional Requirements
✅ Categories
✅ Products PLP loaded from API
✅ Products clickable → Redirect to PDP
✅ Product details loaded dynamically
✅ Header/Footer visible on all pages
